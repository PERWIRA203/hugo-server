Apache Server in Ubuntu

Step by Step :

  1. Update package of your linux
sudo apt-get update
  1. Install package apache
sudo apt-get install apache2
  1. Change Directory to sites-available on directory apache
cd /etc/apache2/sites-available
  1. In this directory, you can see there’s 2 default configuration
perwira203@ubuntu~/etc/apache2/sites-available# ls
000-default.conf  default-ssl.conf
  1. The 000-default.conf is for http service
000-default.conf
  1. The default-ssl.conf is for https service
default-ssl.conf
  1. The configuration that on is the default http service, so you can copy it and make new configuration file
sudo cp 000-default.conf myconfig.conf
  1. Edit your configuration file
sudo nano myconfig.conf
  1. Change the path file of the configuration
DocumentRoot /var/www/html

to

DocumentRoot /var/www/myconfig
  1. Save the file with press ctrl with x, and press y and enter for the changes
  2. Create the directory for our config
sudo mkdir /var/www/myconfig
  1. Change directory to myconfig
cd /var/www/myconfig
  1. Create the html file
sudo touch index.html
  1. edit html file
sudo nano index.html
  1. Write html file
<!DOCTYPE html>
<html>
    <head>
        <title>My Docs file</title>
    </head>
    <body>

        <h1>My Docs file</h1>
        <p>My Docs file</p>

    </body>
</html>
  1. Deactivate the default configuration file
sudo a2dissite 000-default.conf
  1. Activate myconfig configuration file
sudo a2ensite myconfig.conf
  1. Restart your apache server
sudo systemctl reload apache2
  1. This is how it’s gonna look
